# African Ancestry

A modern web prototype inspired by Co–Star’s tone, reimagined for African lineage discovery. Users enter name, birth time, and a preferred tribe to receive a preview and unlock a full ancestral profile.

## What’s inside
- Minimal, high-contrast landing page
- Ancestral profile preview (mocked)
- Full profile unlock flow (mocked)
- Daily guidance snippet

## Run locally
Open `index.html` in your browser.
