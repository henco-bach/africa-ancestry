# Africa Ancestry

A playful, premium web experience inspired by modern gamified UI patterns. Users enter name, birth time, and a preferred tribe to receive a preview and unlock a full ancestral profile.

## What’s inside
- Base44 / Duolingo‑style inspired layout
- Ancestral profile card
- Daily guidance panel
- Profile preview + unlock CTA (mocked)

## Run locally
Open `index.html` in your browser.
