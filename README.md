# African Ancestry

A modern web prototype inspired by Co–Star’s tone, reimagined for African lineage discovery. Users enter name, birth time, and a preferred tribe to receive a personalized ancestral profile and daily guidance.

## What’s inside
- Personal profile generation (mocked)
- Lineage highlights (mocked)
- Daily updates (mocked)
- Clean, modern UI with a lighter aesthetic

## Run locally
Open `index.html` in your browser.
